## Getting Started
### Prerequisites

To clone, 
```
git clone https://github.com/sobia20/target.disease.git
cd gene-info
```
### Run the application

To run the application, execute the following commands,

```bash
yarn install
yarn dev
```
or 
```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the running application.
